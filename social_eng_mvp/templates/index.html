Bem-vindo ao EduPhish

Este projeto simula um ataque de phishing para ensinar sobre segurança online. Configure uma campanha para testar a conscientização dos seus funcionários.

Para começar:

  • Clique abaixo para configurar uma campanha e testar funcionários.
  • Use a simulação para testes locais ou envie e-mails reais com Outlook.
  • Verifique os resultados no dashboard (usuário: admin, senha: admin123).
